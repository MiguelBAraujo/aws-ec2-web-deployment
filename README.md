This project demonstrates a basic cloud infrastructure deployment using Amazon EC2. A web server was manually configured and deployed via SSH.

AWS Services Used:

- Amazon EC2 (t3.micro)
- Amazon Linux 2023
- Security Groups
- SSH
- Apache HTTP Server

Steps Performed:

1. Created an EC2 instance
2. Configured Security Group (SSH + HTTP)
3. Connected via SSH
4. Installed Apache
5. Deployed a custom HTML page

What I Learned:

- How to provision cloud infrastructure
- How to configure secure access
- Basic networking concepts in AWS
- Manual web server deployment
